Etapa 3: Coleta de Treinamento

4 Botoes de Controle Rapido
LOW
LOW: define o modo como LOW — ventilador em velocidade baixa. Todos os dados recebidos passam a ser rotulados com esta classe.
MEDIUM
MEDIUM: define o modo como MEDIUM — ventilador em velocidade media.
HIGH
HIGH: define o modo como HIGH — ventilador em velocidade alta.
PAUSAR
PAUSAR: define o modo como PAUSE. Os dados continuam chegando mas sao rotulados como PAUSE — uteis para marcar intervalos entre coletas.
Badge de modo: o rotulo ativo e exibido como um badge colorido ao lado do titulo da etapa. Verifique sempre antes de mudar a velocidade do ventilador.

Rotulos Avancados (4 Selects)

Alem dos 4 botoes rapidos, o card oferece 4 selects para configuracao avancada de rotulos:

  • Modo principal: LOW / MEDIUM / HIGH / RAW / PAUSE — o rotulo enviado ao servidor.
  • Classe secundaria: ROT_ON / ROT_OFF — para analises de rotacao binaria.
  • Transicao: indica se esta no meio de uma transicao entre classes (flag para descarte).
  • Presets: configuracoes rapidas que setam os 3 selects acima de uma vez.
Sincronizacao: os botoes rapidos (LOW/MED/HIGH/PAUSE) e os selects avancados estao sincronizados. Clicar em "LOW" atualiza automaticamente o select principal para LOW.
Modo Automatico

O modo automatico alterna entre classes por tempo, com transientes automaticos entre cada troca.

Configuracao
  • Selecione quais classes incluir (LOW, MEDIUM, HIGH)
  • Defina o tempo por classe (via seletor)
  • O sistema alterna na sequencia selecionada
Transientes
  • Entre cada troca de classe, um periodo de transiente e descartado
  • Evita dados hibridos (ex.: ventilador acelerando)
  • O badge mostra "TRANSIENTE" durante o descarte
  • Sequencia: as classes sao alternadas na ordem: LOW → MEDIUM → HIGH (ou conforme selecionado).
  • Progresso: uma barra mostra o progresso geral e por classe.
  • Finalizacao: ao concluir todas as classes, o sistema volta para PAUSE automaticamente.
  • Cancelar: voce pode interromper a automacao a qualquer momento.
Dica: o modo automatico e ideal para coletas longas e padronizadas. Para explorar cenarios especificos (ex.: testar uma velocidade marginal), use o modo manual.
Estatisticas da Coleta

O card exibe 4 metricas globais e um breakdown por classe da coleta em andamento.

4 Metricas Globais
  • Total de amostras: contagem geral de leituras rotuladas
  • Tempo de coleta: duracao desde o inicio da sessao
  • Taxa efetiva: amostras/segundo real
  • Ultimo envio: timestamp da ultima leitura recebida
Breakdown por Classe
  • Quantidade de amostras por classe (LOW/MED/HIGH)
  • Percentual de cada classe no total
  • Ajuda a identificar desbalanceamento
  • Dataset equilibrado melhora o classificador
Dataset equilibrado: idealmente, cada classe deve ter quantidade semelhante de amostras. Se uma classe tem muito menos dados, considere coletar mais para ela antes de treinar o modelo.
Manual vs Automatico
Coleta Manual
  • Voce controla quando mudar a classe
  • Ideal para cenarios especificos
  • Flexivel mas requer atencao constante
  • Use o timer (Etapa 4) para padronizar tempos
Coleta Automatica
  • Sistema alterna classes por tempo
  • Transientes descartados automaticamente
  • Ideal para coletas longas e padronizadas
  • Basta configurar, iniciar e esperar
Recomendacao: para a primeira coleta ou experimentos explorativos, use o modo manual. Para coletas de producao (treinar o modelo final), use o modo automatico com pelo menos 5 minutos por classe.
