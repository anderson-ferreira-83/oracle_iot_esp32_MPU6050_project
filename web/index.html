Sobre o Projeto

Sistema IoT completo para monitoramento de vibração de ventiladores industriais em um ambiente local. Utiliza um microcontrolador ESP32 com sensor MPU6050 para coletar dados de vibração a 20Hz. Os dados são enviados via HTTP para um servidor local backend FastAPI + Oracle XE e processados por um modelo de Machine Learning que classifica automaticamente o estado do ventilador em 7 classes (velocidade × estado de rotação).

Após análise comparativa entre 3 algoritmos (Gaussian NB, Random Forest, LightGBM), o Gaussian Naive Bayes foi selecionado por apresentar ~93% de acurácia (CV 5-fold) com 12 features selecionadas via Cohen's d + filtro de correlação para 7 classes, com velocidade de inferência compatível com tempo real.
